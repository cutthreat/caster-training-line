# NOCTA — design notes

## Visual thesis
Северная вода, зелёный камень, лунный кислотный свет и editorial-типографика. Один остров как visual engine.

## Video-derived workflow

- reference roles, not source-copy;
- one hero visual engine;
- one narrative beat per section;
- targeted motion and reveal;
- mobile re-authoring;
- proof before external release.

## Clean-room boundary

NOCTA — самостоятельная интерпретация cinematic-приёмов из публичного видео ADp4isXOva4. Нет копирования его текста, бренда, asset, layout или exact flow.
