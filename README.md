# Линия — product landing

Clean-room product landing inspired by the clarity and scenario-first structure of modern fintech sites.

## Local preview

Open `index.html` directly or serve this folder with any static file server.

## Scope

This is a front-end prototype. The application form shows a local success state and does not send personal data anywhere.
